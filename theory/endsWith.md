### .endsWith( ) method:
This method determines whether a string ends with the character of this string, returning true or false. This method is case-sensitive.<br/><br/>
🐇 Syntax:<br/>
.endsWith(searchString);<br/>
.endsWith(searchString, endPosition);<br/><br/>
<strong>searchString</strong>: the characters to be searched for at the end of the string.It cannot be a regex.<br/><br/>
<strong>endPosition</strong>: this is the end position at which searchString is expected to be found.<br/><br/>
Always the returned values are true and false.<br/>
