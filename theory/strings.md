## Strings

A string object is used to represent and manipulate a sequence of characters.<br/>
Strings are used to hold data that can be represented in a text form.<br/>
Notes to remember:<br/>
We can access a character of a string using <strong>charAt() and the value placed inside the () would be the index position of the value you want to access</strong>.<br/><br/>
But we can also use [] and treat the string as an array so we can use [] and again the index position of the value you want to access.<br/>
Example:<br/>
"cat"[1]; // output "a" <br/>
<strong>Very important:</strong><br/>
🐇 If we want to convert a string into an array => we definitely use the method named <strong>.split("").</strong>
