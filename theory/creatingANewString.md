## Creating a new String
In a codewars exercise i had to solve, it was requested to create a new string replacing some values by 0 and 1. The codewars exercise was named "FakeBinary".<br/>
This is my list of main points to solve the exercise:<br/>
🌼 When creating a new string, we need to have in mind the following:<br/>
🐇 Create a function and the string as argument.<br/>
🐇 Create a variable to store the values for the new string. <br/>
🐇Using a loop which can be a for statement.<br/>
🐇Add conditionals to accomplish the condition requested by the exercise.<br/>
🐇Then using this operator += to add the new value or digit inside the variable created to store the new string.<br/>
🐇After ending the loop and outside the curly braces of the loop, you place the <strong>return</strong>keyword to bring the results for the new string.<br/>
