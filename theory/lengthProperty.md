### Length Property
The length property can be used not only for the arrays but also for the <strong>strings</strong>.
It returns the number of code units used in the string.<br/><br/>
🐇 Note to remember:<br/>
An empty string has as its length the value 0.
<br/><br/>
In the case of the string:<br/>
MDN mentions it contains the length of the string.Here an example:<br/><br/>
const str = "Life, the universe and everything.Answer:";<br/>
console.log(`${str} ${str.length}`);<br/>
Output: "Life, the universe and everything.Answer: 42"
