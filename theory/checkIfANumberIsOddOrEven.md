### How to check if a number is odd or even?
In a codewars exercise I found that there is a way to check if a number is odd or even and it is using <strong>remainder operator</strong>:<br/><br/>
number % 2 === 0 <br/><br/>
This line will verify that all the numbers that are divisible by 2 and have 0 in its remainder are "Even".<br/>
