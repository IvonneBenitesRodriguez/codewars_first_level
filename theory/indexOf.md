## indexOf( ) method:
#### This method of Array instances returns <strong> the first index at which a given element can be found in the array or -1 if it is not present. </strong><br/>
const beasts = ["ant", "bison", "camel", "duck", "bison"];<br/><br/>
console.log(beasts.indexOf("bison"));  // 1 <br/><br/>
