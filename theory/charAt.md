## charAt() 
This method returns the character which belongs to the position entered in charAt().<br/><br/>
Example:<br/><br/>
const name = ["Julia"];
name.charAt(0); // "J" <br/><br/>
🐇Syntax:<br/>
charAt(indexPosition);<br/><br/>
🐇 Note:<br/>
In our exercise, it is helpful to use this method because it returns the first character of each word.<br/>
