## Push method
The push method of an array helps you to add specified elements to the end of an array and returns the new length of the array.<br/><br/>
🐇 It appends values to an array. <br/><br/>
Example:<br/>
const sports = ["soccer","baseball"];<br/>
const allSports = sports.push("basketball", "swimming");
console.log(sports); // ["soccer","baseball","basketball", "swimming"];
